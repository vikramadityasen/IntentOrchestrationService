# AI-ML-Intent-Orchestration-Service
Repository with application code for Intent Orchestration Service for AI/ML

# Installation
Clone this repository & run ```npm install```

# Usage
### For Windows
```bash
env.bat
```
### For Linux
```bash
env.sh
```
Then

```bash
### To create ngrok tunnel 
npm run tunnel

### For debug mode
npm run dev

### For normal execution
npm run start
```
